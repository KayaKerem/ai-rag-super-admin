Genel Layout & Navigasyon

Admin paneli için ana layout yapısını seçelim

A — Collapsed Sidebar (Icon-only)

Vercel tarzı. Sol tarafta dar icon sidebar, geniş content alanı. Temiz, minimal. Hover'da tooltip ile sayfa adı görünür.

B — Full Sidebar (Text + Icon)

GitHub tarzı. Sol tarafta geniş sidebar, icon + text ile navigasyon. Daha okunabilir ama content alanı biraz daralır.

C — Top Navigation

Klasik top bar. Yatay navigasyon, tam genişlikte content. Sayfa az olduğu için yeterli. Ama az sayfa olunca sidebar daha "admin paneli" hissi verir.
